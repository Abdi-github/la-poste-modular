Targeting Guide

📢 Broadcast: Leave all targeting fields empty. Every employee will see the notification.

👤 Individual: Select a specific employee. Only they will see it.

🏷️ By Role: Select ADMIN, BRANCH_MANAGER, or EMPLOYEE. All employees with that role will see it.

🏢 By Branch: Select a branch. All employees assigned to that branch will see it.

🎯 Combined: Set both role and branch to target e.g. all couriers at Zürich branch.
